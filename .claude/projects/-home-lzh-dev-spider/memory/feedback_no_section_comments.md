name: no section header comments

type: feedback
---

Do not add visual guide / section-header comments in code like `// -- A minimal hand-written Task impl --`.

**Why:** The user finds them noisy and unnecessary.

**How to apply:** Never add decorative separator comments or section banners in Rust source files. Let the code structure speak for itself.
